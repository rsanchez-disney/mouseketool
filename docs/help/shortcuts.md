Mouseketool supports keyboard shortcuts for common actions:

| Shortcut | Action |
|---|---|
| `Ctrl + .` | Open the command palette - quick access to any page or action |
| `Ctrl + Enter` | Invoke the selected Lambda from the payload editor (Deployments page) |
| `Escape` | Close the expanded log console or any open modal/dialog |
