Profiles let you load a pre-configured development environment with one click. A profile defines which Lambda projects and batch projects belong to your team's workflow.

Configure profiles from **Settings -> Profile** tab. Select a workspace directory, choose a profile, and click Load.

When a profile is active, a badge appears in the navbar showing the profile name. The Projects page becomes available in the sidebar showing all Lambda projects managed by the profile.
