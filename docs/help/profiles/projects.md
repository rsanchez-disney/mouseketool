The Projects page shows all Lambda projects from the active profile. Each card displays the project name, detected handler class, build tool (Maven/Gradle), and language version.

Cards have three states: **Deployed** (green), **Built - not deployed** (blue), or **Not found** (dashed border). Missing projects can be cloned individually.

Click **Rebuild** to navigate to the Builder page with the project pre-loaded and build auto-started.
